Welcome to the NoCO2 website!

The main objective of this web application is to offer the user the optimal route to minimize CO2 emissions, thus being able to contribute to reducing the impact of the movement of people. With this we seek that each person is aware that there is always some way to contribute however small. Access the application by clicking here.
